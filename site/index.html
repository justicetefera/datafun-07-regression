🧱 Basis

This project began with the CO₂ regression workflow. The Ames dataset (2,930 rows, 83 columns) was used to build a complete regression pipeline.

🛠️ Phase 4 Modifications

Added R² and RMSE functions using scikit‑learn. Customized scatter plots with red, white, and blue colors.

🎯 Phase 5 Custom Project

Extended the workflow to analyze multiple predictors and automatically generate all visualizations.

📄 What I Learned

  • Different predictors have different strengths
  • Residuals reveal model behavior
  • Historic homes disrupt linearity
  • Full workflow skills applied

🏛️ Executive Summary

Overall Quality is the strongest predictor. Size is moderate. Age is weakest due to historic variability. A multivariate model would improve accuracy.
